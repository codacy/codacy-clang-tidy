# clang-analyzer-nullability.NullReturnedFromNonnull

Warns when a null pointer is returned from a function that has
<span id="nonnull">Nonnull</span> return type.

The
<span class="title-ref">clang-analyzer-nullability.NullReturnedFromNonnull</span>
check is an alias, please see [Clang Static Analyzer Available
Checkers](https://clang.llvm.org/docs/analyzer/checkers.html#nullability-nullreturnedfromnonnull)
for more information.
