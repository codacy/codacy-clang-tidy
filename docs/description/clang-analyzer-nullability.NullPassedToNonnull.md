# clang-analyzer-nullability.NullPassedToNonnull

Warns when a null pointer is passed to a pointer which has a
<span id="nonnull">Nonnull</span> type.

The
<span class="title-ref">clang-analyzer-nullability.NullPassedToNonnull</span>
check is an alias, please see [Clang Static Analyzer Available
Checkers](https://clang.llvm.org/docs/analyzer/checkers.html#nullability-nullpassedtononnull)
for more information.
