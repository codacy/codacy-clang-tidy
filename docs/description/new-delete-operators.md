# hicpp-new-delete-operators

This check is an alias for
`misc-new-delete-overloads <../misc/new-delete-overloads>`. Implements
[rule
12.3.1](https://www.perforce.com/resources/qac/high-integrity-cpp-coding-standard/special-member-functions)
to ensure the <span class="title-ref">new</span> and
<span class="title-ref">delete</span> operators have the correct
signature.
