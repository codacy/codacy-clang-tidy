# hicpp-special-member-functions

This check is an alias for
`cppcoreguidelines-special-member-functions <../cppcoreguidelines/special-member-functions>`.
Checks that special member functions have the correct signature,
according to [rule
12.5.7](https://www.perforce.com/resources/qac/high-integrity-cpp-coding-standard/special-member-functions).
